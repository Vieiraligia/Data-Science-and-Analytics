# Data-Science-and-Analytics
📘 Data-Science — Projetos Acadêmicos
por @Vieiraligia

Bem-vindo ao meu repositório de projetos acadêmicos de pós-graduação em Data Science e Analytics da PUC-Rio. Este espaço reúne projetos desenvolvidos no formato MVP (Produto Mínimo Viável), com o objetivo de explorar conceitos, experimentar tecnologias e validar ideias de forma prática e incremental.


Os projetos acadêmicos estão divididos em um MVP para cada sprint.

🚀 <b>MVP 1 — Engenharia de Dados </b>

Este primeiro MVP está associado à disciplina de Engenharia de Dados.
O objetivo é criar um pipeline de dados na plataforma [Databricks](https://databricks.com)

🚀 <b>MVP 2 — Análise de Dados e Boas Práticas </b>

Este  MVP está associado à disciplina de Análise de Dados e Boas Práticas.
O objetivo é produzir um <i>notebook em Python</i> para a realização de análises estatísticas descritivas, visualizações gráficas e pré-processamento de dados.
